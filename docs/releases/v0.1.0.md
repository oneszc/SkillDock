# SkillDock V0.1.0

SkillDock 的第一个可用版本，提供本地 AI Skill 浏览、整理和安装能力。

## 功能

- 扫描 `~/AI-Skills`、`~/.codex/skills` 和 `~/.claude/skills`
- Finder 风格三栏浏览
- 查看 `SKILL.md` 和文件结构
- 编辑独立中文备注，不修改原始 Skill
- 搜索英文内容和中文备注
- 导入本地 Skill
- 安装到 Codex 和 Claude
- 同名冲突保护
- Reveal in Finder 和 Copy Path

## 安装

1. 下载 `SkillDock-0.1.0.zip`
2. 解压后打开 `SkillDock.app`

## 要求与说明

- 需要 macOS 26
- 当前版本使用本地临时签名，尚未进行 Apple Developer ID 签名与公证
- 中文备注保存在 `~/Library/Application Support/SkillDock/`
- 中文备注和未来 AI 总结不会修改原始 Skill
