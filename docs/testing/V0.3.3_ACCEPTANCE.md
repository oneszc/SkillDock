# SkillDock V0.3.3 Acceptance

## Scope

V0.3.3 focuses on manual updates for GitHub-imported Skills and Agent filtering in the second column.

## Checklist

- [ ] GitHub-imported Library Skill shows its remote source in the detail header.
- [ ] Check Update on an unchanged remote Skill reports that the Skill is up to date.
- [ ] Check Update on a changed remote Skill opens a preview with added, modified, and removed files.
- [ ] Local modified Skill is marked as requiring review and cannot be replaced silently.
- [ ] Confirming an available update replaces the Library Skill files.
- [ ] Confirming an available update updates the stored source commit and content hash.
- [ ] Library second column defaults to All Agents.
- [ ] Library can filter Skills installed in Codex.
- [ ] Library can filter Skills installed in Claude.
- [ ] Installed can filter Skills installed in Codex.
- [ ] Installed can filter Skills installed in Claude.
- [ ] System does not show the Agent filter.
