# SkillDock V0.4 Acceptance

## Checklist

- [ ] Existing Codex and Claude settings migrate correctly.
- [ ] Library scans enabled dynamic Agent targets.
- [ ] Disabled Agent targets are not scanned.
- [ ] Detail header shows enabled Agent targets dynamically.
- [ ] Install Targets list shows enabled Agent targets dynamically.
- [ ] Installing to a custom Agent copies the Skill into that Agent path.
- [ ] Uninstalling from a custom Agent removes only that Agent copy.
- [ ] Library filter can filter by a custom Agent.
- [ ] Installed filter can filter by a custom Agent.
- [ ] Skill list rows show installed Agent logos dynamically.
- [ ] System page remains read-only.
- [ ] System page does not show Agent filter.
- [ ] Settings can enable and disable Agent targets.
- [ ] Settings can edit Agent target paths.
- [ ] Suggested Agent targets can be added disabled by default.

## Manual Notes

- Suggested Agent targets should not scan until the user enables them.
- Codex and Claude remain enabled by default for existing users.
- V0.4 does not include batch sync, symlink mode, background sync, or AI Chinese notes.
