- [x] Existing Codex and Claude settings migrate correctly.

- [x] Disabled Agent targets are not scanned.
- [x] Detail header shows enabled Agent targets dynamically.
- [x] Install Targets list shows enabled Agent targets dynamically.

- [x] Uninstalling from a custom Agent removes only that Agent copy.
- [x] Library filter can filter by a custom Agent.
- [x] Installed filter can filter by a custom Agent.
- [x] Skill list rows show installed Agent logos dynamically.
- [x] System page remains read-only.
- [x] System page does not show Agent filter.
- [x] Settings can enable and disable Agent targets.
- [x] Settings can edit Agent target paths.
- [x] Suggested Agent targets can be added disabled by default.
- [x] Suggested Agent targets show product-provided colored and gray brand logos.
