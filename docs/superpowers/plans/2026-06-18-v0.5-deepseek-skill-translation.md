# V0.5 DeepSeek Skill Translation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add secure, manual DeepSeek-powered Chinese translation for one Skill's description and `SKILL.md` without modifying the original Skill.

**Architecture:** Keep provider networking, credentials, translation persistence, and SwiftUI state separate. `SkillDockCore` owns provider-neutral models, DeepSeek request mapping, Keychain access, translation storage, and matching; `SkillDockApp` owns settings and detail-page presentation through provider-neutral AppModel operations.

**Tech Stack:** Swift 6.3, SwiftUI, Observation, Foundation `URLSession`, macOS Security/Keychain, XCTest, existing JSON storage and Markdown preview components.
