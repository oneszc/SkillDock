# V0.2.1 Visual Polish Design

## Direction

SkillDock 使用 App Store 的舒适阅读感和 Finder 的管理效率。

本次不改变 Skill 管理功能，只统一主要页面的字号、间距、阅读宽度和设置页结构。

## Rules

- 列表主标题使用更清晰的强调层级，副标题不再使用过小的 caption。
- Skill 列表行增加垂直留白，同时保持可快速扫描。
- 详情页标题约 32pt，英文和中文描述使用更舒适的阅读字号。
- 详情页主要内容左右留白提升到约 28pt。
- 长文内容限制阅读宽度，避免在宽窗口中形成过长文本行。
- 表单、导入预览和设置页统一增加内容留白。
- 进入 Settings 后隐藏无意义的 Skill 列表中间栏，改为“设置分类 + 设置内容”双栏结构。
- 设置分类当前只显示 General，为后续设置分组保留清楚结构。
- General 顶部提供 System、Light、Dark 三种外观模式，选择后立即影响整个应用并保存。
- Appearance 使用一整块系统灰色圆角底板。
- Appearance 标题位于底板左上角，三个模式选项在底板右侧横向排列。
- 模式预览使用产品负责人提供的 System、Light、Dark 图片，保持原始比例。
- 模式名称显示在图片下方；选中项使用系统强调色描边，名称加粗。
- 继续使用 macOS 原生控件、系统背景和 SF Symbols。

## Boundaries

- 不使用 App Store 式营销卡片。
- 普通 Skill 浏览继续使用 Finder 式三栏结构。
- 不引入自定义字体、第三方图标或固定颜色主题。
- 不调整 Skill 数据和原始 Skill 内容。
