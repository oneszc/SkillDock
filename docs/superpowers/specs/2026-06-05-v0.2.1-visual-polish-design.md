# V0.2.1 Visual Polish Design

## Direction

SkillDock 使用 App Store 的舒适阅读感和 Finder 的管理效率。

本次不改变功能、信息结构和交互流程，只统一主要页面的字号、间距和阅读宽度。

## Rules

- 列表主标题使用更清晰的强调层级，副标题不再使用过小的 caption。
- Skill 列表行增加垂直留白，同时保持可快速扫描。
- 详情页标题约 32pt，英文和中文描述使用更舒适的阅读字号。
- 详情页主要内容左右留白提升到约 28pt。
- 长文内容限制阅读宽度，避免在宽窗口中形成过长文本行。
- 表单、导入预览和设置页统一增加内容留白。
- 继续使用 macOS 原生控件、系统背景和 SF Symbols。

## Boundaries

- 不使用 App Store 式营销卡片。
- 不改变 Finder 式三栏结构。
- 不引入自定义字体、第三方图标或固定颜色主题。
- 不调整 Skill 数据和原始 Skill 内容。
