01

One Skill Library

See your main library, installed Skills, and system Skills in one Finder-like workspace.

02

Local + GitHub Import

Import a local Skill folder or public GitHub repository with preview and conflict protection.

03

Safe Agent Sync

Install or remove exact Skill copies across configurable Agent targets without touching your source library.

04

Manual Update Checks

Install

Two steps. No package manager.

SkillDock is currently distributed as a ZIP file through GitHub Releases.

01

Download & Unzip

Open the latest GitHub Release, download SkillDock-0.5.0.zip, then unzip it.

02

Move & Launch

Move SkillDock.app to Applications, open it, then configure the Agent targets you use in Settings.

FAQ

Questions, answered clearly.

Codex and Claude Code are enabled by default. You can also configure Gemini, OpenCode, Grok, Antigravity, Hermes, and custom Agent Skill directories.

SkillDock copies a selected Skill from your main library to the Agent directories you choose, with conflict checks and safe removal.

Skill files, notes, and settings stay local. Only the optional translation feature sends the selected SKILL.md content to DeepSeek using your own API key.

SkillDock requires macOS 26 or later.

Yes. SkillDock is free and MIT-licensed open source.
